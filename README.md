# Project Pitch Timer

A visual countdown timer designed for project pitches with color-coded phases.

## Features

- 6-minute total timer (4 min pitch + 2 min Q&A)
- Color-coded phases:
  - Green: Opening phase
  - Yellow: Transition to Q&A
  - Orange: Q&A phase
  - Red: Time's up with flashing alert
- Circular progress ring visualization
- Start/Pause/Reset controls
- Add extra time (+1 min) when timer ends
- Responsive design

## Usage

Simply open `index.html` in your browser to start using the timer.

### Controls

- **Start/Pause**: Begin or pause the countdown
- **Reset**: Return to initial 6:00 state
- **+1 min**: Add extra time (only available when timer reaches 0:00)

## Timer Phases

1. **Project Pitch** (4 minutes): Present your project
2. **Q & A** (2 minutes): Answer questions

The interface automatically transitions between phases and updates the visual theme accordingly.
